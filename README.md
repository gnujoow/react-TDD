# react TDD

this repo is personal practice for react TDD(Test driven development). the test is built with  
- [MochaJS](https://mochajs.org/)
- [Chai-jquery](https://github.com/chaijs/chai-jquery)
- [Chai](http://chaijs.com/api/bdd/)

to run app

```
$ npm start
```

to run test

```
$ npm run test:watch
```
